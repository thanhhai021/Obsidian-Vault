Lỗi 500 Internal Server Error

Nguyên nhân:

- Lỗi cú pháp trong `.htaccess`.
    
- PHP script bị lỗi.
    
- Thiếu extension PHP.
    

Cách khắc phục:

- Check log error:  
    Apache: `/var/log/apache2/error.log`  
    Nginx: `/var/log/nginx/error.log`
    
- Tắt từng dòng trong `.htaccess` để dò lỗi.
    
- Kiểm tra PHP version và bật đủ module.
